软件

安装PyCharm

本人使用的软件是PyCharm的免费版，

需要在 [https://www.jetbrains.com/pycharm/] 里下载并安装

PyCharm中文插件

安装完PyCharm后整个界面是英语，所以需要安装中文插件

在菜单栏找到file中Settings...或着使用快捷键Ctrl+Alt+s打开设置

打开后在Plugins里搜索 中文 ，找到并安装该插件，安装完成后应用并重启软件就好了

安装Chrome和驱动器

本人使用Chrome来进行自动化测试，所以整篇笔记是只适用Chrome浏览器的自动化

安装完Chrome后需要使用Chrome驱动器
